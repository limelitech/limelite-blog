Creating a personal profile website is one of the best ways to showcase your work, skills, and online presence. In this tutorial, I’ll walk you through the step-by-step process of building a modern and responsive profile website using **HTML**, **CSS**, and **JavaScript** — and then deploying it using **GitHub Pages**.

## 🧰 What You Need

- A code editor (VS Code recommended)

- A GitHub account

## 📁 Folder Structure

Here’s a quick look at how your project folder should be organized:

```
profile-website/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── images/
│   └── profile.jpg
```

---

## 🧱 Step 1: Create the HTML Structure

```html
<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Your Name - Profile</title>
  <link rel="stylesheet" href="css/style.css" />
</head>
<body>
  <header>
    <img src="images/profile.jpg" alt="Profile Photo" class="profile-img" />
    <h1>Your Name</h1>
    <p>Web Developer | Designer | Creator</p>
  </header>

  <section class="about">
    <h2>About Me</h2>
    <p>This is a short bio about yourself...</p>
  </section>

  <footer>
    <p>© 2025 Your Name</p>
  </footer>

  <script src="js/script.js"></script>
</body>
</html>
```

---

## 🎨 Step 2: Style with CSS

```css
/* css/style.css */
body {
  font-family: Arial, sans-serif;
  background-color: #f7f7f7;
  text-align: center;
  margin: 0;
  padding: 0;
}

.profile-img {
  width: 150px;
  border-radius: 50%;
  margin-top: 30px;
}

header {
  background: #333;
  color: white;
  padding: 40px 0;
}
```

---

## ✨ Step 3: Add Interactivity (Optional)

```js
// js/script.js
console.log("Welcome to my profile website!");
```

---

## 📦 Step 4: Host It on GitHub Pages

1. Push your code to a GitHub repository.
2. Go to **Settings > Pages** in the repo.
3. Set the source to your `main` branch, and select the root directory.
4. Save and wait a few minutes — your site will be live!

## 📁 Folder Structure

Here is the complete folder structure from the downloadable ZIP:

```
profile-website-master/
├── assets/
│   ├── css/
│   │   └── style.css
│   ├── js/
│   │   └── script.js
│   └── img/
│       └── profile.jpg
├── index.html
├── README.md
```

Each part of this structure serves a purpose:

- **index.html** – The main structure of your website.
- **assets/css/style.css** – All your custom styling.
- **assets/js/script.js** – Optional JavaScript functionality.
- **assets/img/profile.jpg** – Your profile photo.

---

## 🧱 Step 1: Build the HTML Structure

The `index.html` file is where the core content of your website lives.

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>John Doe - Profile</title>
  <link rel="stylesheet" href="assets/css/style.css">
</head>
<body>
  <header>
    <img src="assets/img/profile.jpg" alt="Profile Photo" class="profile-img">
    <h1>John Doe</h1>
    <p>Web Developer | Tech Enthusiast | Creator</p>
  </header>

  <section class="about">
    <h2>About Me</h2>
    <p>Hello! I’m John, a passionate web developer who enjoys building clean and responsive websites. This profile showcases who I am and what I do.</p>
  </section>

  <footer>
    <p>&copy; 2025 John Doe</p>
  </footer>

  <script src="assets/js/script.js"></script>
</body>
</html>
```

## 🚀 Step 4: Deploy with GitHub Pages

Once your site is ready, it’s time to share it with the world.

### Step-by-step GitHub Deployment

1. **Create a GitHub repository** (e.g., `profile-website`).
2. **Push your project** to the repository:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/your-username/profile-website.git
git push -u origin main
```

3. **Enable GitHub Pages**:
   - Go to your repository on GitHub.
   - Navigate to **Settings > Pages**.
   - Under **Source**, select the `main` branch and root directory (`/`).
   - Click **Save**.

After a few minutes, your site will be live at `https://your-username.github.io/profile-website/`.
